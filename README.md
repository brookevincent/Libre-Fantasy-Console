# Libre Fantasy Console

This is a fantasy console based on a computer from the late 
1980s - early 1990s. Hardware-wise, it will take inspiration
from the Macintosh Classic, while software-wise it will 
borrow from both Unix-like and Windows operating systems.

## Planned "Hardware" Specifications

This fantasy console will have:

- Keyboard and mouse input
- 640 x 480 display with 8-bit color
- 4MB of memory
- 256KB of video memory
- 40MB of storage

All files on the disk of the fantasy console will be
accessible to the user, so exporting programs the user makes
will be as simple as copying and pasting.

## Planned Software Specifications

This fantasy console will run off of a custom bytecode 
similar to that used by the Lua interpreter. This will be
loaded into the console's similated memory during run time 
to run programs on the console.

Included by default on the fantasy console will be a
termainal, a code/ text editor, an image manipulator, a
programming language compiler, as well as a suite of basic
software that can be removed easily.
